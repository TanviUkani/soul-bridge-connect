Soul Bridge - Volunteer Matching Platform

🌐 Soul Bridge

Soul Connect is a web platform designed to connect volunteers with NGOs based on skills, interests, and availability. It helps users find meaningful opportunities and allows organizations to manage volunteers efficiently.

🚀 Features

👤 User Authentication (Login / Signup)
🤝 Volunteer & NGO Matching
📅 Availability & Scheduling
📊 Impact Tracking
📝 Testimonials & Feedback
💾 Database Integration (Supabase / Backend)
⚡ Fast frontend using modern frameworks
🛠️ Tech Stack

Frontend

HTML, CSS, JavaScript
React (Vite)
Tailwind CSS

Backend

Node.js / Flask (based on your setup)
REST API

Database

Supabase / PostgreSQL / SQLite
